# ⚡ BrazilVoltz

> Plataforma Open Source para monitoramento, gerenciamento e otimização do carregamento de veículos elétricos.

## 📖 Sobre o Projeto

O BrazilVoltz é uma plataforma digital desenvolvida para auxiliar motoristas, empresas e gestores de frotas no acompanhamento do desempenho energético de veículos elétricos.

A solução centraliza informações sobre monitoramento de bateria, consumo energético, custos de recarga, histórico de abastecimento, localização de estações de carregamento e análise de desempenho, oferecendo uma visão integrada da mobilidade elétrica.

O projeto foi desenvolvido com base nos princípios do Software Livre e Open Source, incentivando a colaboração da comunidade e a evolução contínua da plataforma.

---

## 🎯 Problema que o Projeto Resolve

Com o aumento da utilização de veículos elétricos, usuários e empresas enfrentam dificuldades para acompanhar informações importantes relacionadas ao consumo de energia, custos operacionais e planejamento de recargas.

Muitas dessas informações encontram-se dispersas em diferentes aplicativos e plataformas, dificultando o gerenciamento eficiente dos veículos.

O BrazilVoltz busca centralizar esses recursos em uma única solução, proporcionando maior controle, economia e praticidade para seus usuários.

## 🚀 Funcionalidades do MVP

### Recursos Gratuitos

* Monitoramento do nível da bateria;
* Estimativa de autonomia do veículo;
* Histórico de recargas;
* Relatórios básicos de abastecimento;
* Cálculo inteligente de custos de utilização;
* Estimativa de gastos mensais;
* Alertas de carregamento.

### Recursos Premium

* Mapa inteligente de carregadores próximos;
* Disponibilidade de estações em tempo real;
* Comparação de preços entre eletropostos;
* Rotas inteligentes para recarga;
* Relatórios avançados de desempenho;
* Diagnóstico da saúde da bateria;
* Monitoramento ambiental (CO₂ economizado);
* Dashboard corporativo;
* Controle de frotas empresariais;
* Monitoramento em nuvem.

## 📦 Instalação e Execução

O BrazilVoltz encontra-se atualmente na fase de MVP (Minimum Viable Product), disponibilizado como protótipo demonstrativo desenvolvido em HTML.

### Requisitos

* Navegador moderno (Google Chrome, Mozilla Firefox, Microsoft Edge ou Opera);
* Arquivos do projeto baixados localmente.

### Passo 1 – Obter o Projeto

```bash
git clone https://github.com/WDguiz/Brasilvoltz_MVP.git
```

### Passo 2 – Abrir o Projeto

Localize o arquivo:

```text
index.html
```

### Passo 3 – Executar

Abra o arquivo em qualquer navegador moderno.
